# Button-W

Пропсы:

- `address: string` - отвечает за адресс кошелька пользователя.

За что отвечает:

- Кнопка-контроля кошелька

Где используется?

- Header

```ts
<button-w
  [address]="{адрес кошелька}"
>
```

Функционал

- При клике открывается меню управления кошельком
- Копирование кошелька
- Открытие в tonviewer
- Отключение кошелька
